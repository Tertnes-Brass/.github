# Tertnes Brass

Tertnes Brass er et varmt og inkluderende brassband fra Tertnes i Bergen. Korpset ble stiftet 23. januar 1964 og spiller i Elitedivisjonen i NM Brass.

Denne GitHub-organisasjonen samler våre digitale tjenester og åpne prosjekter.

## Prosjekter

- [tertnesbrass](https://github.com/Tertnes-Brass/tertnesbrass) – nettsiden vår med konserter, nyheter og informasjon om korpset.
- [tb-notearkiv](https://github.com/Tertnes-Brass/tb-notearkiv) – notearkiv og medlemsplattform for arkiv, prosjekter og vikardeling.
